# Usage
<!-- usage -->
```sh-session
$ npm install -g zerotier-central-cli
$ ztc COMMAND
running command...
$ ztc (-v|--version|version)
zerotier-central-cli/0.0.0 darwin-x64 node-v11.10.1
$ ztc --help [COMMAND]
USAGE
  $ ztc COMMAND
...
```
